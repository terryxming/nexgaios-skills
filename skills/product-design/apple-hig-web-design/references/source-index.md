## 提炼原则

- 把 Apple HIG 当设计指导，不当 Web CSS 代码。
- 只有能服务用户任务时，才把平台规则转成 Web 等价实现。
- 原则和行为优先于视觉模仿。
- SF Symbols 只做概念参考；在 Web 项目中优先使用已有 icon library，不假定有授权资源。
- 面向公众的 Web 页面避免暗示产品由 Apple 制作。
