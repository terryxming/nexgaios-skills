# 参考资料

本目录用于存放 `skill-doctor` 的领域规则、资料路由、协议说明和维护备注。

资料路由：

- `audit-rubric.md`：审计、挑刺、诊断、改造建议时读取。
- `lifecycle-policy.md`：发布检查、版本治理、CHANGELOG、冻结、回滚、复盘时读取。
- `report-format.md`：需要输出正式报告、表格、门禁结论或复盘模板时读取。
- `test-matrix.md`：需要设计测试、验证候选版本或 forward-test 时读取。

维护要求：

- 文件名使用小写 kebab-case。
- 每个引用文件只服务一个明确主题。
- `SKILL.md` 中必须说明何时读取对应引用文件。
